# 質因數分解

輸入一個數字，把它分解成質數的乘積，如：`50 = 2 ∗ 5^2`

## Input

從鍵盤輸入一個正整數 `n`, `n <= 2^32 - 1`

## Output

輸出 `n` 的質因數分解，格式請參考 `Sample Output`

## Sample

1.  Input

    ```
    10
    ```

    Output

    ```
    2 * 5
    ```

2.  Input

    ```
    60
    ```

    Output

    ```
    2^2 * 3 * 5
    ```

3.  Input

    ```
    1024
    ```

    Output

    ```
    2^10
    ```

4.  Input

    ```
    1073741826
    ```

    Output

    ```
    2 * 3 * 59 * 3033169
    ```

5.  Input

    ```
    4294967295
    ```

    Output

    ```
    3 * 5 * 17 * 257 * 65537
    ```

## Hint

簡單的質因數分解法：n 為欲分解之正整數，i 為變數 (從 2 開始)

1.  i 若能整除 n 則分解，直到不能整除為止。

    此時 n = 45，i = 2。

2.  若不能整除則嘗試下一個數，重複步驟一。

    此時 n = 15，i = 3。

    此時 n = 5，i = 3。

3.  一直重複前兩個步驟，直到 i = n 時，結束。

    此時 n = 5，i = 4。

    此時 n = 5，i = 5，結束。
