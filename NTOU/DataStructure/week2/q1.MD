# 選擇排序法

由使用者輸入數值個數 n ，由程式把這 n 個數值從小到大排好。

## Input

輸入每次共兩行，第一行為要排序的數值總數，第二行則為需要排序的數字，以空格隔開。

## Output

輸出為排序之後的結果，如輸入為 `5 7 2` 則輸出之排序為 `2 5 7`。

## Sample

1.  Input

    ```
    3
    55 20 37
    ```

    Output

    ```
    20 37 55
    ```

2.  Input

    ```
    3
    5 2 7
    ```

    Output

    ```
    2 5 7
    ```
