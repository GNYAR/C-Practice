# 多項式加法與乘法

多項式運算，利用 linked-list 的模式建立兩條多項式，並且進行加法運算以及乘法運算，再將所運算的結果輸出。

## Input

第一參數為係數，第二參數為指數，輸入的係數如為 -1 則多項式建立完成。必須建立兩次多項式。

- 指數與係數皆小於 10
- 係數為正數
- 相同的指數不重複輸入

## Output

將運算的加法跟乘法顯示出來，詳細格式參照範例測資。

## Sample

Input

```
3 3
1 1
2 2
4 4
-1
2 2
3 3
1 0
-1
```

Output

```
add = 4x^4 + 6x^3 + 4x^2 + 1x + 1
mult = 12x^7 + 17x^6 + 12x^5 + 11x^4 + 5x^3 + 2x^2 + 1x
```
