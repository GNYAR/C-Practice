# 稀疏矩陣乘法

`A` 是一個 `n x n` 的稀疏矩陣，`A'`是
`A` 的轉置矩陣，計算 `A x A'`。(n < 10000)

## Input

第一行為一個整數(指接者輸入的行數)。最多為 10 。

接下來的形式皆為 `(row:col)=val`，代表在此稀疏矩陣的第幾列第幾行有一個非零值 val 。行數與列數由 1 開始起算。

## Output

印出 `A x A'` 的稀疏矩陣內所有不為 0 的元素，格式為 `row col value` 並且先以 row 排序再以 col 排序

每行的最後僅有換行，沒有空白

## Sample

1.  Input

    ```
    3
    (3:2)=3
    (2:3)=2
    (1:2)=2
    ```

    Output

    ```
    1 1 4
    1 3 6
    2 2 4
    3 1 6
    3 3 9
    ```

2.  Input

    ```
    6
    (6:2)=6
    (1:3)=9
    (8:7)=3
    (8:2)=9
    (1:5)=-5
    (7:3)=1
    ```

    Output

    ```
    1 1 106
    1 7 9
    6 6 36
    6 8 54
    7 1 9
    7 7 1
    8 6 54
    8 8 90
    ```
